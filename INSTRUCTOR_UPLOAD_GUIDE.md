# Instructor Upload Guide - Quick Reference

## üé¨ Uploading Videos

### Steps:

1. Go to your course ‚Üí Select a topic
2. Click **"Add Material"** button
3. Select **"Video"** from Material Type dropdown
4. Fill in:
   - **Title** (e.g., "Python Basics Lecture")
   - **Description** (optional)
5. Click file input and select your video
   - ‚úÖ **Maximum size: 500MB**
   - ‚úÖ Supported formats: MP4, WebM, OGG, etc.
6. Click **"Upload Material"** button
7. Wait for upload (large files may take several minutes)

### Tips:

- For videos > 500MB, compress using Handbrake or similar
- Use H.264 codec for best compatibility
- Recommended: 1080p @ 30fps with bitrate 2-5 Mbps

## ‚ùì Uploading MCQ Quizzes

### Steps:

1. Go to your course ‚Üí Select a topic
2. Find the **"Quiz"** section
3. Click **"Upload MCQ"** button (blue button next to "Generate")
4. Fill in quiz details:
   - **Quiz Title** \* (required)
   - **Description** (optional)
   - **Difficulty Level**: Beginner / Intermediate / Advanced
   - **Duration**: Minutes (e.g., 15 or 30)
5. Create questions:
   - Enter question text in the textarea
   - Fill in 4 options (Option A, B, C, D)
   - Select the correct answer using radio buttons
6. To add more questions: Click **"Add Question"** button
7. To remove a question: Click **"Remove"** button
8. Click **"Upload MCQ Quiz"** when done

### Tips:

- Write clear, unambiguous questions
- Ensure all 4 options are filled
- Make sure exactly ONE correct answer is selected
- Recommended: 5-10 questions per quiz
- Include time limit appropriate for difficulty

     - Any webpage: https://example.com/...
5. Click **"Upload Material"** button

## ‚ö†Ô∏è Common Errors & Solutions

### Error: "File too large"

**Cause:** File exceeds size limit
**Solution:**

- Video: Reduce file size to < 500MB
- PDF: Reduce file size to < 50MB
- Use compression tools or split into multiple files

### Error: "Upload timeout"

**Cause:** Network too slow or file very large
**Solution:**

- Check internet connection
- Try again with a smaller file
- Try uploading at a different time
- Maximum wait: 10 minutes

### Error: "Failed to upload material"

**Cause:** Server error or network issue
**Solution:**

- Try again
- Check file format
- Ensure file isn't corrupted
- Contact admin if persists

### Error: "Invalid materialType"

**Cause:** Wrong file format selected
**Solution:**

- Select correct material type
- Only VIDEO, PDF, or LINK
